1、使用文件类 ： File类（更新中）

如何获取文件信息

方法 说明 返回值类型
getName() 获取文件名称 String
getParent() 获取文件的父路径字符串 String
getPath() 获取文件的相对路径字符串 String
getAbsolutePath() 获取文件的绝对路径字符串 String
exists() 判断文件或者文件夹是否存在 boolean
isFile() 判断是不是文件类型 boolean
isDirectory() 判断是不是文件夹类型 boolean
delete() 删除文件或文件夹，如果删除成功返回结果为true否则false,删除时文件夹内必须为空 boolean
mkdir() 创建文件夹，创建成功返回true boolean
setReadOnly() 设置文件或文件夹的只读属性 boolean
length() 获取文件的长度 long
lastModified() 获取文件的最后修改时间 long
list() 获取文件夹中的文件和子文件夹的名称，并存放到字符串数组中 String[]
